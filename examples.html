Portal Automation & Reporting System

Quick Stats:
85% process automation  •  75% reduction in manual errors

Before

  • Manual navigation across multiple portals for data extraction
  • Separate tools for report generation and email distribution
  • Repetitive manual tasks prone to errors
  • No centralized real-time progress tracking or logging

After

  • Unified PyQt6 GUI automates portal logins and data extraction
  • Integrated Playwright-based browser automation for efficient downloads
  • Real-time progress updates and error logging via multi-threaded workers
  • Automated report generation, file archival, and dynamic dashboard display

Leveraging Python, PyQt6, and Playwright, this solution integrates data extraction, reporting, and real‑time progress tracking into a single application—reducing manual intervention and ensuring accurate, timely outputs.
